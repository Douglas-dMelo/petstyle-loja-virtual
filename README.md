# 🐾 Pet&Style - Loja Virtual para Pets

![HTML](https://img.shields.io/badge/HTML5-orange)
![TailwindCSS](https://img.shields.io/badge/TailwindCSS-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-yellow)
![Status](https://img.shields.io/badge/status-em%20desenvolvimento-green)

Projeto de uma **Landing Page de e-commerce fictício para pets**, desenvolvido para praticar **HTML, Tailwind CSS e JavaScript** aplicando conceitos modernos de desenvolvimento front-end.

O projeto simula a interface inicial de uma **loja virtual estilizada para pets**, com layout responsivo e funcionalidades interativas.

# 🧠 Funcionalidades

✔ Página inicial responsiva  
✔ Layout mobile-first  
✔ Cards de produtos dinâmicos  
✔ Modal de visualização de produto  
✔ Carrinho lateral estilo e-commerce  
✔ Contador automático de itens no carrinho  
✔ Cálculo automático do preço total  
✔ Avaliações de produtos com estrelas  
✔ Animações utilizando Tailwind CSS
